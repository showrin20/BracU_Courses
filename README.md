## ğŸš€ Getting Started

### Prerequisites
- Python 3.x
- Jupyter Notebook (for .ipynb files)
- GCC Compiler (for C programs)
- Git

### Installation & Usage

1. **Clone the repository:**
```bash
git clone https://github.com/showrin20/BracU_Courses.git
cd BracU_Courses
```

2. **For Python assignments:**
```bash
cd CSE110  # or any course directory
python assignment_file.py
```

3. **For Jupyter notebooks:**
```bash
jupyter notebook CSE470.ipynb
# or
jupyter notebook MAT120.ipynb
```

4. **For C programs:**
```bash
cd CSE321
gcc program.c -o program
./program
```
